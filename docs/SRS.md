1. User Friendliness
The system shall allow a new user to complete registration within 2 minutes without external help.

2. Data Security
The system shall encrypt all student records using AES-256 and restrict access to authenticated users only.

3. System Performance
The system shall load the dashboard page within 3 seconds under normal network conditions (≤100ms latency).

4a. Registration
The system shall allow users to register by providing name, email, and password.

4b. Login
The system shall allow users to log in using email and password.

4c. Profile Management
The system shall allow users to update profile fields including name, password, and profile picture.

5. Notifications
The system shall send an email notification to users within 10 seconds after successful registration.
